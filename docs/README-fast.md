## 语音合成极速版API
### 中英文语音极速合成版本
* 100毫秒极速识别
* 支持中英文混合文本
* 支持感情表达
#### URL
**`https://asr-pre.abcpen.com:8443/tts/v1/fast/gen`**
中英文的语音合成

#### KEY
* 使用之前，请向商务申请appKey和appSecret, 以正常服务请求。
#### 请求参数：

* 以HTTPS POST(**json**)请求发送

|    参数    | 数据类型 | 是否必须 | 说明                                                         | 默认值    |
| :--------: | -------- | -------- | :----------------------------------------------------------- | --------- |
